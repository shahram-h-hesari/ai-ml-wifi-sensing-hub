# Code Releases

This folder tracks code repositories, implementation resources, baseline models, and reproducibility notes related to AI/ML WiFi sensing.

## Code tracking goals

| Field | Purpose |
|---|---|
| Repository | GitHub, GitLab, project page, or official source |
| Related Paper | Paper connected to the code release |
| Language / Framework | Python, MATLAB, PyTorch, TensorFlow, etc. |
| Dataset Requirement | Dataset needed to run the code |
| Setup Clarity | Whether environment, dependencies, and instructions are clear |
| Reproducibility Notes | Barriers, missing files, unclear steps, or successful reproduction |
| License | Open-source license or usage limitation |

## Purpose

The goal is to identify which AI/ML WiFi sensing papers provide usable code and which research gaps remain due to missing, incomplete, or difficult-to-reproduce implementations.
