# Collaboration

This folder tracks research groups, authors, labs, datasets, open-source projects, and communities related to AI/ML WiFi sensing.

## Candidate types

| Candidate Type | Examples |
|---|---|
| Authors | Researchers publishing highly relevant papers |
| Labs | Groups working on WiFi sensing, healthcare AI, wireless security, or aging technology |
| Dataset Owners | People or groups maintaining useful datasets |
| Code Maintainers | Developers maintaining reproducible implementations |
| Research Communities | AI/ML healthcare, wireless sensing, cybersecurity, IoT, and smart-environment communities |

## Purpose

The goal is to support structured scientific collaboration by connecting papers, datasets, code, and open research gaps to people and groups working in related areas.
